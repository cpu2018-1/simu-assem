形式について

./main BFILE [IFILE] [option]	([]は任意)
BFILE	:出力用ファイルの名前
IFILE	:入力用ファイルの名前
option	:-dのみ




デバッグモード(-d)のコマンドについて

<移動用>
n(next)		:次の命令に移ります
j(jump) OPECODE	:次に(OPECODE)命令が来るところまで移ります(OPECODEは小文字)
l(last) 	:全命令が終了するところまで移ります
q(quit)		:プログラムを終了します

<表示用>
reg		:汎用レジスタの表を出力します
freg		:浮動小数点レジスタの表を出力します
mem N		:メモリ4*N番地の値をintで表示します
pc		:プログラムカウンタを表示します
io		:IOの値を表示します(現状IOにほぼ対応していないので注意)
